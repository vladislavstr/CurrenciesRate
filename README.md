# CurrenciesRate

---
### Web service
Based on the daily data on the exchange rates of the Central Bank (https://www.cbr-xml-daily.ru/daily_json.js)

- [ ] <u><strong>GET /currencies</strong></u> — returns a list of currency rates with the possibility of pagination
- [ ] <u><strong>GET /currency/</strong></u> — returns the currency exchange rate for the transmitted currency identifier
